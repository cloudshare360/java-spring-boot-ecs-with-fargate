# 01 - Spring Boot App

High-level steps:
- Scaffold or open the HelloWorld Spring Boot service.
- Implement a simple HTTP GET endpoint (e.g., `/`) returning a static message.
- Build locally with Maven or Gradle (e.g., `mvn clean package` or `./gradlew build`).
- Verify local run (`mvn spring-boot:run` or `java -jar target/*.jar`) and confirm the endpoint responds.
